🖥 VPS Deployment

```bash
git clone https://github.com/THE-VIP-BOY-OP/VIP-MUSIC.git
cd VIP-MUSIC
bash setup
nano .env  # Fill your variables
sudo apt install tmux && tmux
bash start
```

📌 Pro Tip: To detach from tmux – press Ctrl+B then D

⚙️ CONFIGURATION

🔐 Required Variables

· API_ID – Get from my.telegram.org
· API_HASH – Get from my.telegram.org
· BOT_TOKEN – Get from @BotFather
· MONGO_DB_URI – MongoDB connection string
· LOG_GROUP_ID – Log channel/group ID
· OWNER_ID – Your Telegram user ID
· STRING_SESSION – Pyrogram v2 string session

🌟 Premium Add-ons (Optional)

· SPOTIFY_CLIENT_ID – Spotify API client ID
· SPOTIFY_CLIENT_SECRET – Spotify API client secret
· GOOGLE_CLOUD_API_KEY – (https://console.cloud.google.com/apis/api/youtube.googleapis.com)
  · 📈 Quota: 10,000 requests per day (free tier)
· HEROKU_API_KEY – For Heroku automation
· HEROKU_APP_NAME – Your Heroku app name

🔗 GOOGLE CLOUD API SETUP GUIDE

📌 Step-by-Step:

1. Visit: Google Cloud Console - YouTube API
2. Create a project (or select existing)
3. Enable YouTube Data API v3
4. Go to Credentials → Create Credentials → API Key
5. Copy your API Key and add to .env as GOOGLE_CLOUD_API_KEY
6. Quota Details:
   · ✅ Free Tier: 10,000 requests per day
   · ⚠️ Rate Limits: Apply for increased quota if needed
